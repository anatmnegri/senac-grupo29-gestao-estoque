# Sistema de Gestão de Estoque - Grupo 29 (Senac)

Este é o **Projeto Integrador** desenvolvido pelo Grupo 29 para o curso de Análise e Desenvolvimento de Sistemas do **Senac EAD**. O objetivo da aplicação é oferecer uma solução prática e eficiente para o controle de produtos e inventário em pequenas empresas.

---

## 🛠️ Tecnologias Utilizadas

A solução foi construída utilizando uma stack moderna e leve:

* **Linguagem:** JavaScript / Node.js
* **Servidor:** [servidor.js](servidor.js) (Executando a lógica de API e rotas)
* **Banco de Dados:** [SQLite](https://www.sqlite.org/) (Persistência local no arquivo `estoque.db`)
* **Front-end:** HTML, CSS e JavaScript estáticos (localizados na pasta `/publico`)
* **Gerenciamento de Dependências:** NPM

---

## 📂 Organização do Repositório

* `/publico`: Contém os arquivos de interface (front-end).
* `servidor.js`: Ponto de entrada da aplicação.
* `database.js`: Lógica de conexão e criação das tabelas no banco de dados.
* `estoque.db`: Arquivo do banco de dados relacional.
* `package.json`: Configurações do projeto e dependências instaladas.
